# Accusaga Enterprise Semiconductor AI Demo

Classic business-level Streamlit application for demonstrating Accusaga's AI-driven semiconductor transformation capability.

## Features
- Executive cockpit
- Real-time semiconductor operations simulator
- Yield prediction
- Defect anomaly detection
- Thermal hotspot intelligence
- Timing risk intelligence
- Root-cause analysis
- ROI impact dashboard
- Accusaga capability mapping
- Client problem-to-solution view
- Executive PDF report generation

## Setup

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python scripts_generate_data.py --records 5000
streamlit run dashboard/app.py
```

## Demo Login
No login required for easy client demo.
